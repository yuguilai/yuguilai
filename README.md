# 📖 项目简介

`yuguilai` 团队于 **2024 年 2 月** 启动的项目（现已停更并关闭阅读权限）。

项目主要用于：

- 研究 QQ 个性装扮资源结构
- 分析 QQ 个性装扮商城资源规律
- 体验 QQ 个性装扮中的付费及门槛装扮
- 研究 Android QQ9 的相关实现逻辑
